# Beginner Python Projects Portfolio 🚀
**By Zayaan**

A collection of Python scripts and mini-projects developed during my first month of learning programming, logic building, and CLI application design.

---

## 📦 Projects Included

1. **`first calculator.py`** — Basic command-line calculator performing standard arithmetic operations (`+`, `-`, `*`, `/`).
2. **`Advance CALC.py`** — Enhanced CLI calculator featuring continuous operation loops and `try/except` error handling for invalid user inputs.
3. **`guess the number.py`** — Interactive number-guessing game using Python's `random` module, guess tracking, and dynamic range validation.
4. **`cave adventure.py`** — Text-based adventure game structured with reusable functions, conditional branching, and decision paths.
5. **`Gamer slang dictionary(dict prac).py`** — Interactive dictionary tool using key-value mapping, string normalization (`.strip().lower()`), and user-driven entry additions.
6. **`hangman.py`** — Word-guessing game featuring difficulty selection (Easy/Hard word banks), life tracking, and input validation.
7. **`password generator.py`** — Customizable password generator leveraging Python's `string` and `random` standard library modules.
8. **`rock paper scissors.py`** — Classic player vs. computer game implementing fundamental conditional checks and game reset loops.
9. **`rps ai 2.py`** — Refactored, high-quality version of Rock-Paper-Scissors with structured boolean win conditions and clean UI output.
