# My-PYTHON-grind-so-far-(1-month)
I'm Zayaan,a student tryna learn to code amidst study pressure. This repository contains the projects I built during my first month of learning Python. Everything was written from scratch to focus on the underlying logic.

## The Projects

**hangman.py** - A modular Hangman game with difficulty                        modes, guess tracking, and input validation.
**cave_adventure.py** - A choice-based text adventure                                 structured using separate functions                           for story nodes.
**advance_calculator.py** - A calculator that uses try/except                             blocks to handle errors and zero-                             division.
**rock_paper_scissors.py** - Play against the computer,                                    featuring input protection to                                 prevent crashes.
**password_generator.py** - Uses random and string libraries                              to generate passwords based on                                user-defined length.
**number_generator.py** - A number guessing game using loops                            and boundary limits.
